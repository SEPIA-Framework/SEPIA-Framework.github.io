Since SEPIA is self-hosted, privacy focused, open-source assistant there is no global server that will store your data.

To learn how you can create your own server for example using a Raspberry Pi SBC please visit: SEPIA documentation

If you are the administrator of your own server please visit the SEPIA Control-HUB to create new user accounts.


If you got this link from your company or a friend that hosts his own SEPIA server please ask them how to get your account.
